# Dependencies
There are no dependencies to install

# Getting Started

#### Create object

`Notification notification = new Notification();`

#### Object customization 

`notification.Set*;`

#### Show object

`notification.Show();`

### Hello World
```
Notification notification = new Notification();
notification.SetContent("Hello World");
notification.Show();
```
